Garbage Classification

Developed and compared YOLO V6, V7, and V8 models for garbage classification using Python and PyTorch. Implemented transfer learning to enhance model performance, with extensive testing revealing YOLO V7 as the most precise version. The project involved generating results with confidence scores on bounding boxes during the testing phase. Concluded that YOLO V7 outperformed other versions in precision, providing a robust solution for effective waste management. Implementing this project in the real world can significantly improve waste sorting efficiency, reduce human error, and promote recycling efforts.
